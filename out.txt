      Параметры компилятора Visual Basic

                                  - ВЫХОДНОЙ ФАЙЛ -
/out:<файл>                   Задает имя выходного файла.
/target:exe                       Создать консольное приложение (по умолчанию). (Краткая форма: /t)
/target:winexe                    Создать приложение для Windows.
/target:library                   Создать библиотечную сборку.
/target:module                    Создать модуль, который может быть добавлен в сборку.
/target:appcontainerexe           Создать приложение Windows, выполняемое в контейнере AppContainer.
/target:winmdobj                  Создать промежуточный файл метаданных Windows
/doc[+|-]                         Создает XML-файл документации.
/doc:<file>                       Создает XML-файл документации <file>.

                                  - ВХОДНЫЕ ФАЙЛЫ -
/addmodule:<список_файлов>
                                  Указывать метаданные из заданного модуля.
/link:<список_файлов> Внедрять метаданные из указанной сборки взаимодействия. (Краткая форма: /l)
/recurse:<подстановочный
                                  Включает все файлы в текущем каталоге и подкаталогах в соответствии с заданным шаблоном.
/reference:<список_файлов>
                                  Указывать метаданные из заданной сборки. (Краткая форма: /r)

                                  - РЕСУРСЫ -
/linkresource:<сведения_о_рес
                                  Компонует указанный файл в качестве внешнего ресурса сборки. resinfo:<файл>[,<имя>[,public|private]] (Краткая форма: /linkres)
/nowin32manifest                  Манифест по умолчанию не должен внедряться в раздел манифеста выходного PE.
/resource:<сведения_о_рес
                                  Добавляет указанный файл в качестве внедренного ресурса сборки. resinfo:<файл>[,<имя>[,public|private]] (Краткая форма: /res)
/win32icon:<файл>             Задает файл значка Win32 (.ico) для ресурсов Win32 по умолчанию.
/win32manifest:<файл>         Предоставленный файл внедрен в раздел манифеста выходного PE.
/win32resource:<файл>         Задает файл ресурсов Win32 (.res).

                                  - ГЕНЕРИРОВАНИЕ КОДА -
/optimize[+|-]                    Включить оптимизацию.
/removeintchecks[+|-]             Удалить целочисленные проверки. По умолчанию отключены.
/debug[+|-]                       Выдавать отладочные сведения.
/debug:full                       Выдавать полные отладочные сведения (по умолчанию).
/debug:pdbonly                    Выдавать только файл PDB.

                                  - ОШИБКИ И ПРЕДУПРЕЖДЕНИЯ -
/nowarn                           Отключить предупреждения.
/nowarn:<список_чисел> Отключить список индивидуальных предупреждений.
/warnaserror[+|-]                 Обрабатывать предупреждения как ошибки.
/warnaserror[+|-]:<список_чисел>
                                  Трактовать список предупреждений как ошибки.

                                  - ЯЗЫК -
/define:<список_символо
                                  Объявить глобальные символы условной компиляции. symbol_list:name=value,... (Краткая форма: /d)
/imports:<список_импорта
                                  Объявить глобальные импорты для пространств имен в указанных файлах метаданных. import_list:namespace,...
/langversion:<номер>         Укажите версию языка: 9|10|11.
/optionexplicit[+|-]              Требовать явного объявления переменных.
/optioninfer[+|-]                 Разрешить вывод для типов переменных.
/rootnamespace:<строка>     Задает корневое пространство имен для всех объявлений типов.
/optionstrict[+|-]                Требовать строгой семантики языка.
/optionstrict:custom              Предупреждать, когда не накладывается ограничение на строгую семантику языка.
/optioncompare:binary             Задает сравнение строк как двоичных данных. Задано по умолчанию.
/optioncompare:text               Задает сравнение строк как текста.

                                  - РАЗНОЕ -
/help                             Отображает это сообщение о синтаксисе. (Краткая форма: /?)
/noconfig                         Файл VBC.RSP не включается в состав автоматически.
/nologo                           Не отображать заставку компилятора со сведениями об авторских правах.
/quiet                            Режим вывода без сообщений.
/verbose                          Отображать подробные сообщения.

                                  - ДОПОЛНИТЕЛЬНО -
/baseaddress:<номер>         Базовый адрес библиотеки или модуля (шестнадцатеричный).
/bugreport:<файл>             Создать файл отчета об ошибках.
/codepage:<номер>            Указывает кодовую страницу, используемую при открытии исходных файлов.
/delaysign[+|-]                   Использовать отложенную подпись для сборки, используя только открытую часть ключа строгого имени.
/errorreport:<строка>       Указывает способ обработки внутренних ошибок транслятора; должен принимать значения prompt, send, none или queue (по умолчанию)
/filealign:<номер>           Указывает выравнивание для разделов выходного файла.
/highentropyva[+|-]               Включить ASLR с высокой энтропией.
/keycontainer:<строка>      Задает контейнер ключа для строгого имени.
/keyfile:<файл>               Задает файл ключа для строгого имени.
/libpath:<список_путей>Список каталогов для поиска ссылок на метаданные. (Разделитель — точка с запятой.)
/main:<класс>                Задает класс или модуль, содержащий Sub Main. Он может являться производным классом от System.Windows.Forms.Form. (Краткая форма: /m)
/moduleassemblyname:<строка>Имя сборки, частью которого будет данный модуль.
/netcf                            Целевая версия .NET Compact Framework.
/nostdlib                         Не обращаться к стандартным библиотекам (system.dll и VBC.RSP).
/platform:<строка>          Ограничения на платформы, в которых может исполняться данный код; допустимые значения: x86, x64, Itanium, arm, AnyCPU32BitPrefe
/sdkpath:<путь>               Местоположение каталога .NET Framework SDK (mscorlib.dll).
/subsystemversion:<версия>  Укажите версию подсистемы выходного PE-файла. version:<номер>[.<номер>]
/utf8output[+|-]                  Выдавать выходные данные компилятора в кодировке UTF8.
@<файл>                       Вставить параметры командной строки из текстового файла.
/vbruntime[+|-|*]                 Скомпилируйте с помощью или без использования стандартной среды исполнения Visual Basic.
/vbruntime:<file>                 Скомпилируйте при помощи альтернативной среды исполнения Visual Basic в <файл>.
